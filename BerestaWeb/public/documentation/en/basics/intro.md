# Introduction

Beresta Language is a lightweight, expressive scripting language designed for game logic, automation, and embedded systems. It combines the simplicity of GameMaker Language (GML) with the clarity of C-style syntax.

---

## Main Features
- Simple, readable syntax.
- Support for variables, expressions, functions, and control flow.
- Built-in math, array, and matrix functions.
- Dynamic typing: numbers, strings, booleans, arrays, and enums.
- Easy integration into custom engines and interpreters.
