# Операторы и выражения

Операторы в Beresta Language определяют, как значения комбинируются, сравниваются или преобразуются.  
Язык поддерживает арифметические, сравнительные, логические и операторы присваивания, подобные тем, что используются в языках семейства C.

---

## Арифметические операторы

| Оператор | Описание | Пример | Результат |
|-----------|-----------|----------|-----------|
| `+` | Сложение | `2 + 3` | 5 |
| `-` | Вычитание | `10 - 4` | 6 |
| `*` | Умножение | `3 * 4` | 12 |
| `/` | Деление | `10 / 2` | 5 |
| `%` | Остаток от деления (modulo) | `10 % 3` | 1 |
| `-x` | Унарный минус | `-5` | -5 |

Пример:
```beresta
let a = 5;
let b = 2;
console_print(a * b + 10); // 20
```

---

## Операторы сравнения

| Оператор | Описание | Пример | Результат |
|-----------|-----------|----------|-----------|
| `==` | Равно | `1 == 1` | true |
| `!=` | Не равно | `1 != 2` | true |
| `>` | Больше чем | `1 > 2` | false |
| `<` | Меньше чем | `1 < 2` | true |
| `>=` | Больше или равно | `1 >= 1` | true |
| `<=` | Меньше или равно | `1 <= 1` | true |

Пример:
```beresta
let x = 10;
let y = 20;

if(x < y)
{
    console_print("x is smaller"); // x меньше
}
```

---

## Логические операторы

Beresta поддерживает как символические (`&&`, `||`), так и словесные (`and`, `or`) логические операторы.  
Они могут использоваться взаимозаменяемо для повышения читаемости кода.

Пример:
```beresta
let a = 5;
let b = 10;

if(a < b and b < 20)
{
    console_print("Both conditions true"); // Оба условия истинны
}

if(!(a > b))
{
    console_print("Negation works"); // Отрицание работает
}
```

---

## Операторы присваивания

| Оператор | Описание | Пример | Результат |
|-----------|-----------|----------|-----------|
| `=` | Присваивание | `a = 5` | 5 |
| `+=` | Прибавление и присваивание | `a += 5` | 10 |
| `-=` | Вычитание и присваивание | `a -= 5` | 5 |
| `*=` | Умножение и присваивание | `a *= 5` | 25 |
| `/=` | Деление и присваивание | `a /= 5` | 5 |
| `%=` | Остаток и присваивание | `a %= 5` | 0 |

Пример:
```beresta
let a = 10;
a += 5;
a *= 2;
console_print(a); // 30
```

---

## Приоритет операторов

Beresta вычисляет выражения в следующем порядке (от высшего к низшему):

- Скобки ()
- Унарные операторы: !, -
- Умножение, деление, остаток от деления
- Сложение, вычитание
- Операторы сравнения
- Логические операторы
- Присваивания =, +=, -= и т. д.

Пример:
```beresta
let x = 2 + 3 * 4;            // 14
let y = (2 + 3) * 4;          // 20
if(!(x > y or y > 0)) {}      // false
```

---

## Примечания

- Деление между двумя целыми числами возвращает результат с плавающей точкой.  
- Логические операторы всегда возвращают логическое значение (`true` или `false`).  
- Можно смешивать `and`/`or` с символическими операторами, однако для лучшей читаемости рекомендуется придерживаться одного стиля.
