# Переменные

Переменные в Beresta Language имеют динамическую типизацию. Они могут хранить числа, строки, логические значения, массивы и другие значения. Ключевое слово `let` при объявлении переменной является необязательным.

---

## Синтаксис
```beresta
let name = value;
```
или просто:
```beresta
name = value;
```

---

## Описание

Переменные можно объявлять с использованием или без использования ключевого слова `let`. Если переменной присваивается значение без `let`, она автоматически создаётся в текущей области видимости.

Все переменные являются динамическими и не требуют явного указания типа.

---

## Примеры

### Базовое присваивание
```beresta
let a = 10;
let b = 5;
let c = a + b;
console_print(c); // 15
```

### Без использования `let`
```beresta
x = 100;
y = 25;
console_print(x - y); // 75
```

### Строки и конкатенация
```beresta
name = "User";
console_print("Hello, " + name);
```

### Логические флаги
```beresta
is_ready = true;
if(!is_ready)
{
    console_print("Not ready yet!"); // Ещё не готов!
}
else
{
    console_print("Ready to go!"); // Готов к запуску!
}
```

---

## Примечания

- Все переменные изменяемы.
- Повторное объявление переменной перезаписывает её предыдущее значение.
- Логические значения поддерживают операторы `!`, `&&`, `||`, `and`, `or`.
