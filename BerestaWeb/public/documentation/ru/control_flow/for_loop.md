# For Цикл

Цикл `for` повторяет блок кода заданное количество раз, используя переменную-счётчик, которая изменяется на каждой итерации.  
Он идеально подходит, когда заранее известны значения начала, конца и шага.

---

## Синтаксис
```beresta
for(initialization; condition; increment)
{
    // код, выполняемый на каждой итерации
}
```

---

## Описание

Цикл `for` состоит из трёх основных частей:

- Инициализация — выполняется один раз в начале (например, `let i = 0`).  
- Условие — проверяется перед каждой итерацией. Если оно ложно, цикл завершается.  
- Инкремент — выполняется после каждой итерации (например, `i = i + 1`).  

Цикл продолжается, пока условие остаётся истинным.

---

## Примеры

Простой счётчик:
```beresta
for(let i = 0; i < 3; i = i + 1)
{
    console_print("Loop iteration: " + i); // Итерация цикла: ...
}
```

Использование `continue` и `break`:
```beresta
for(let j = 0; j < 10; j = j + 1)
{
    if(j % 2 == 0) {continue;}  // пропустить чётные числа
    if(j == 7)     {break;}     // остановиться, когда j достигнет 7

    console_print("Odd number: " + j); // Нечётное число: ...
}

console_print("for loop finished"); // цикл for завершён
```

Цикл без `let`:
```beresta
for(i = 0; i < 5; i = i + 1)
{
    console_print(i);
}
```

---

## Примечания

- Все выражения в заголовке цикла `for` вычисляются по порядку.  
- Можно объявлять переменные внутри цикла с помощью `let`.  
- `break` немедленно завершает выполнение цикла.  
- `continue` пропускает текущую итерацию и переходит к следующей.
