# Foreach Цикл

Цикл `foreach` перебирает все элементы массива или вложенной коллекции.  
Он упрощает чтение и обработку данных без необходимости вручную управлять индексом.

---

## Синтаксис
```beresta
foreach(element in array)
{
    // код, использующий 'element'
}
```

---

## Описание

Оператор `foreach` извлекает каждый элемент из массива или вложенного массива и присваивает его временной переменной на каждой итерации.

Он автоматически обрабатывает длину массива, делая перебор данных чище и менее подверженным ошибкам по сравнению с ручным циклом `for`.

Можно использовать вложенные циклы `foreach` для обхода многомерных массивов.

---

## Примеры

Базовые итерации:
```beresta
let arr = [10, 20, 30, "text", [5, 15, 25], false];

foreach(x in arr)
{
    console_print(x);
}
```

Вложенные массивы:
```beresta
let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

foreach(row in matrix)
{
    foreach(val in row)
    {
        console_print(val);
    }
}
```

Использование условий и управляющих операторов:
```beresta
foreach(n in [5, 10, 15, 20])
{
    if(n > 10)
    {
        console_print("Above 10: " + n); // Больше 10: ...
    }
}
```

Прерывание и пропуск внутри вложенных циклов:
```beresta
let nested_matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

foreach(row in nested_matrix)
{
    foreach(val in row)
    {
        if(val == 5)
        {
            console_print("skip 5"); // пропустить 5
            continue;
        }

        if(val == 8)
        {
            console_print("stop inner at 8"); // остановить внутренний цикл на 8
            break;
        }

        console_print(val);
    }

    if(row[0] == 7)
    {
        console_print("stop outer loop"); // остановить внешний цикл
        break;
    }
}

console_print("foreach loop done"); // цикл foreach завершён
```

---

## Примечания

- Переменная цикла является временной копией каждого элемента.  
- `break` завершает текущий цикл `foreach`.  
- `continue` пропускает текущий элемент и переходит к следующему.  
- Работает рекурсивно с вложенными массивами.  
- Изменение исходного массива внутри цикла поддерживается, но может повлиять на результат итерации.
