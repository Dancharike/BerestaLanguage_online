# If / Else

Оператор `if` позволяет выполнять код в зависимости от того, истинно или ложно заданное условие.  
Он может комбинироваться с `else` или `else if`, чтобы создавать сложные ветвления логики.

---

## Синтаксис
```beresta
if(condition)
{
    // код, когда условие истинно
}
else if(other_condition)
{
    // код, когда первое условие ложно, но это истинно
}
else
{
    // код, когда все условия ложны
}
```

---

## Описание

Оператор `if` вычисляет логическое выражение.  
Если результат истинный, выполняется блок кода, следующий за `if`.  
Если нет — управление передаётся следующему `else if` или `else` (если они присутствуют).

Условия могут использовать логические (`and`, `or`, `!`) и сравнительные операторы.

---

## Примеры

Проверка простого условия:
```beresta
let a = 10;
let b = 20;

if(a < b)
{
    console_print("a is smaller than b"); // a меньше b
}
else
{
    console_print("a is greater or equal to b"); // a больше или равно b
}
```

Вложенные if-операторы:
```beresta
x = 5;
y = 10;

if(x > y)
{
    console_print("x is greater"); // x больше
}
else if(x < y)
{
    z = x + y;
    console_print("sum: " + z); // сумма: ...
}
else
{
    console_print("x equals y"); // x равен y
}
```

Использование логических операторов:
```beresta
a = 5;
b = 10;

if(a < b and b < 20)
{
    console_print("Both conditions true"); // Оба условия истинны
}

if(!(a > b))
{
    console_print("Negation example"); // Пример отрицания
}
```

---

## Примечания

- Фигурные скобки `{}` обязательны для блоков кода.  
- `else if` должен следовать сразу после `if` или другого `else if`.  
- Условия вычисляются динамически; допустимы как числовые, так и логические выражения.  
- Логические операторы могут записываться как символами (`&&`, `||`, `!`), так и словами (`and`, `or`).
