# Repeat Цикл

Цикл `repeat` выполняет блок кода фиксированное количество раз.  
Он полезен, когда заранее известно, сколько итераций нужно выполнить.

---

## Синтаксис
```beresta
repeat(count)
{
    // код, выполняющийся несколько раз
}
```

---

## Описание

Оператор `repeat` выполняет заключённый блок ровно `count` раз.  
Значение `count` вычисляется до начала выполнения цикла.  
Если `count` равно нулю или отрицательно, тело цикла не выполняется.

В отличие от `while`, этот цикл не проверяет условие на каждой итерации — он просто выполняет заданное количество повторов.

---

## Примеры

Базовое повторение:
```beresta
repeat(5)
{
    console_print("Iteration"); // Итерация
}
```

Случайные числа:
```beresta
repeat(5)
{
    console_print(int_random_range(1, 10)); // вывод случайного числа от 1 до 10
}
```

Использование `break` внутри `repeat`:
```beresta
let random_check = 0;

repeat(10)
{
    let n = int_random_range(1, 10);
    console_print("num: " + n); // число: ...

    if(n == 7)
    {
        console_print("found 7, stopping early"); // найдено 7, остановка досрочно
        break;
    }

    random_check = random_check + 1;
}

console_print("repeat done, total iterations: " + random_check); // цикл repeat завершён, всего итераций: ...
```

---

## Примечания

- Циклы `repeat` всегда начинаются с нуля и выполняются заданное количество раз.  
- Переменная-счётчик является неявной (недоступна напрямую).  
- Внутри цикла можно использовать `break` и `continue`.  
- Если значение количества повторов дробное, оно округляется вниз до ближайшего целого.
