# While Цикл

Цикл `while` повторяет блок кода до тех пор, пока заданное условие остаётся истинным.  
Он полезен для выполнения операций, когда количество итераций заранее неизвестно.

---

## Синтаксис
```beresta
while(condition)
{
    // код, выполняющийся, пока условие истинно
}
```

---

## Описание

Перед каждой итерацией проверяется условие в скобках.  
Если оно возвращает `true`, выполняется тело цикла.  
Когда условие становится `false`, выполнение цикла прекращается.

Важно, чтобы условие в какой-то момент изменилось, иначе цикл будет выполняться бесконечно.

---

## Примеры

Простой счётчик:
```beresta
i = 0;

while(i < 5)
{
    console_print(i);
    i = i + 1;
}

console_print("Loop done!"); // Цикл завершён!
```

Использование `break` и `continue`:
```beresta
let counter = 0;

while(counter < 10)
{
    counter = counter + 1;

    if(counter == 3)
    {
        console_print("skip 3"); // пропустить 3
        continue;
    }

    if(counter == 8)
    {
        console_print("stop at 8"); // остановиться на 8
        break;
    }

    console_print("counter: " + counter); // счётчик: ...
}

console_print("while done"); // цикл while завершён
```

---

## Бесконечный цикл

Цикл, в котором условие никогда не становится ложным, будет выполняться бесконечно.  
Убедись, что логика цикла изменяет переменные, участвующие в условии.

Пример бесконечного цикла (не рекомендуется):
```beresta
while(true)
{
    console_print("This never ends"); // Это никогда не закончится
}
```

---

## Примечания

- Условие `while` проверяется перед каждой итерацией.  
- Используй `break`, чтобы немедленно выйти из цикла.  
- Используй `continue`, чтобы перейти к следующей итерации.  
- Тело цикла может содержать любые допустимые операторы, включая вложенные циклы или условия.
