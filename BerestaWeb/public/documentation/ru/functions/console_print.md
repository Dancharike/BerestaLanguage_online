# Вывод консоли

Функция `console_print()` выводит текст или значения в консоль во время выполнения программы.  
Она используется в основном для отладки, отображения результатов или отслеживания выполнения программы.

---

## Синтаксис
```beresta
console_print(value);
```

Можно также объединять несколько значений или строк:
```beresta
console_print("x = " + x + ", y = " + y);
```

---

## Описание

`console_print()` преобразует переданное значение в строку и выводит его в консоль программы.  
Она поддерживает все примитивные типы данных, массивы и сложные структуры.

Каждый вызов автоматически добавляет новую строку после вывода.

---

## Примеры

Простой вывод:
```beresta
console_print("Hello from BerestaLanguage!"); // Привет из BerestaLanguage!
```

Вывод чисел и выражений:
```beresta
let a = 5;
let b = 3;

console_print(a + b);                // 8
console_print("Result: " + (a * b)); // Результат: 15
```

Вывод массивов:
```beresta
let arr = [10, "hi", true, [1, 2, 3]];
console_print(arr);
```

Вывод матриц:
```beresta
matrix_identity();
console_print(matrix_get());
```

Вывод строк с переменными:
```beresta
let name = "Denis";
let score = 95;

console_print("Player " + name + " scored " + score + " points!"); // Игрок Denis набрал 95 очков!
```

---

## Примечания

- Автоматически добавляет перевод строки после вывода.  
- Несколько аргументов нужно объединять вручную с помощью `+`.  
- Массивы и матрицы выводятся рекурсивно со всеми вложенными элементами.  
- Можно напрямую выводить результат функции:
```beresta
console_print(sqrt(25)); // 5
```
