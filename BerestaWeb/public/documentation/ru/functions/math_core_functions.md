# Базовая Математика

Базовая математическая библиотека **BerestaLanguage** предоставляет фундаментальные числовые операции: округление, логарифмы, ограничение значений, интерполяцию и статистические функции.  
Все эти функции работают с числами (`double` и `int`) и автоматически проверяют корректность аргументов во время выполнения.

Если передан некорректный аргумент, интерпретатор выдаёт диагностическое сообщение об ошибке с указанием файла и строки.

---

## Список функций

| Функция | Описание |
|-----------|-----------|
| `sqr(x)` | Возвращает квадрат числа `x`. |
| `sqrt(x)` | Возвращает квадратный корень числа `x`. Аргумент должен быть неотрицательным. |
| `abs(x)` | Возвращает абсолютное (неотрицательное) значение `x`. |
| `round(x)` | Округляет `x` до ближайшего целого. |
| `floor(x)` | Округляет `x` вниз (в сторону отрицательной бесконечности). |
| `ceil(x)` | Округляет `x` вверх (в сторону положительной бесконечности). |
| `frac(x)` | Возвращает дробную часть числа (`x - floor(x)`). |
| `power(x, y)` | Возводит `x` в степень `y`. |
| `clamp(value, min, max)` | Ограничивает значение в пределах от `min` до `max`. |
| `lerp(a, b, t)` | Выполняет линейную интерполяцию между `a` и `b` по коэффициенту `t` (0.0 → a, 1.0 → b). |
| `min(a, b)` | Возвращает меньшее из двух чисел. |
| `max(a, b)` | Возвращает большее из двух чисел. |
| `mean(a, b, c, …)` | Возвращает среднее арифметическое всех аргументов (требуется хотя бы один аргумент). |
| `median(a, b, c, …)` | Возвращает медиану из всех аргументов (среднее значение при сортировке). Требуется хотя бы один аргумент. |
| `ln(x)` | Возвращает натуральный логарифм `x` (основание *e*). Допустимо только для `x > 0`. |
| `log2(x)` | Возвращает логарифм `x` по основанию 2. Допустимо только для `x > 0`. |
| `log10(x)` | Возвращает логарифм `x` по основанию 10. Допустимо только для `x > 0`. |
| `logn(base, value)` | Возвращает логарифм числа `value` по основанию `base`. Оба аргумента должны быть положительными, и `base ≠ 1`. |

---

## Описание

`sqr(x)` Возвращает квадрат числа:
```beresta
console_print(sqr(5)); // 25
```

`sqrt(x)` Возвращает квадратный корень числа:
```beresta
console_print(sqrt(9)); // 3
```

`abs(x)` Возвращает абсолютное значение (расстояние от нуля):
```beresta
console_print(abs(-4.5)); // 4.5
```

`round(x)`, `floor(x)`, `ceil(x)` — функции округления, аналогичные большинству языков:
```beresta
console_print(round(3.6)); // 4
console_print(floor(3.6)); // 3
console_print(ceil(3.1));  // 4
```

`frac(x)` Возвращает дробную часть числа, убирая целую:
```beresta
console_print(frac(3.14)); // 0.14
```

`power(x, y)` Возводит `x` в степень `y`:
```beresta
console_print(power(2, 10)); // 1024
```

`clamp(value, min, max)` Ограничивает значение в заданных пределах:
```beresta
console_print(clamp(17, 0, 10)); // 10
console_print(clamp(-5, 0, 10)); // 0
```

`lerp(a, b, t)` Выполняет линейную интерполяцию между `a` и `b`:
```beresta
console_print(lerp(0, 100, 0.3)); // 30
```

`min(a, b)` и `max(a, b)` Возвращают меньшее или большее из двух чисел соответственно:
```beresta
console_print(min(10, 3)); // 3
console_print(max(10, 3)); // 10
```

`mean(a, b, c, …)` Вычисляет среднее значение всех аргументов:
```beresta
console_print(mean(1, 2, 3, 4)); // 2.5
```

`median(a, b, c, …)` Сортирует все аргументы и возвращает средний элемент. Если количество аргументов чётное, используется нижнее среднее значение:
```beresta
console_print(median(1, 9, 3, 7)); // 5
```

`ln(x)`, `log2(x)`, `log10(x)` — все логарифмические функции требуют положительного аргумента:
```beresta
console_print(ln(25));       // 3.218
console_print(log2(1024));   // 10
console_print(log10(10000)); // 4
```

`logn(base, value)` Вычисляет логарифм числа `value` по заданному основанию:
```beresta
console_print(logn(5, 25)); // 2
```

---

## Примечания

- Все аргументы автоматически приводятся к числовому типу (`double` и `int`).  
- Несоответствие типов или недопустимые значения вызывают диагностическую ошибку времени выполнения.  
- Ни одна из функций не изменяет переменные — все возвращают новые вычисленные значения.  
- Арифметические выражения можно свободно вкладывать, например: `sqrt(power(a, 2) + power(b, 2))`.
