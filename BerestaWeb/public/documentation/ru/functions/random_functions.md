# Случайности

BerestaLanguage предоставляет лёгкую систему псевдослучайных чисел, используемую во множестве встроенных функций.

Все функции, связанные со случайностью, относятся к этой группе:

| Функция | Описание |
|-----------|-----------|
| `choose_from(a, b, c, …)` | Возвращает один случайный элемент из всех переданных аргументов. |
| `random(max)` | Возвращает случайное вещественное число от `0.0` (включительно) до `max` (не включая). |
| `random_range(min, max)` | Возвращает случайное вещественное число в диапазоне от `min` до `max` (включительно). |
| `int_random(max)` | Возвращает случайное целое число от `0` до `max` (включительно). |
| `int_random_range(min, max)` | Возвращает случайное целое число от `min` до `max` (включительно). |

---

## Описание

`choose_from(a, b, c, …)` Выбирает один случайный элемент из переданного списка аргументов. Поддерживаются все типы данных (числа, строки, логические значения, массивы и т. д.):
```beresta
console_print(choose_from(10, 20, 30, 40)); // выведет 10, 20, 30 или 40
console_print(choose_from("apple", "orange", "banana"));
```

`random(max)` Генерирует число с плавающей точкой в диапазоне от `0.0` до `max`:
```beresta
console_print(random(1));   // 0.0 ≤ x < 1.0
console_print(random(10));  // 0.0 ≤ x < 10.0
```

`random_range(min, max)` Генерирует число с плавающей точкой в диапазоне от `min` до `max`:
```beresta
console_print(random_range(5, 15));  // 5.0 ≤ x ≤ 15.0
console_print(random_range(10, 5));  // также допустимо, значения меняются местами автоматически
```

`int_random(max)` Генерирует целое число от `0` до `max` (включительно):
```beresta
console_print(int_random(5)); // возможные значения: 0, 1, 2, 3, 4, 5
```

`int_random_range(min, max)` Генерирует целое число в диапазоне `[min, max]`:
```beresta
console_print(int_random_range(5, 15)); // 5 ≤ x ≤ 15
console_print(int_random_range(10, 2)); // также допустимо, значения меняются местами автоматически
```

---

## Примечания

- `random()` и `random_range()` используют равномерное распределение для вещественных чисел (`double`).  
- `int_random()` и `int_random_range()` используют равномерное распределение для целых чисел.  
- Начальное значение генератора задаётся системой при запуске, поэтому результаты различаются между сессиями.  
- Все функции чистые с точки зрения логики — они не изменяют переменные, а только возвращают вычисленные значения.
