# Встроенные постоянные

BerestaLanguage предоставляет несколько предопределённых **макросов (констант)**, которые доступны глобально из любого файла проекта.  
Они не требуют объявления или импорта и могут использоваться напрямую в выражениях, функциях или выводе в консоль.

---

## Обзор

Макросы — это постоянные значения, представляющие полезные глобальные данные: версию движка, математические константы или предопределённые идентификаторы.  
Они вычисляются при запуске, но хранятся как неизменяемые значения.

---

## Объявление

Макрос можно объявить простым образом:
```beresta
#macros VERSION   = "1.0.0";
#macros PI        = 3.14159;
```
Рекомендуется создать отдельный `.beresta` файл, в котором будут храниться все макросы проекта.

---

## Пример

```beresta
console_print("Running version: " + VERSION);

function area_of_circle(r)
{
    return PI * r * r;
}

console_print("Circle area: " + area_of_circle(5));
```

Результат выполнения:
```beresta
Running version: 1.0.0
Circle area: 78.5398
```

---

## Примечания

- Макросы идеально подходят для указания версии, математических констант и глобальных настроек проекта.
